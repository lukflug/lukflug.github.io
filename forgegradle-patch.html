Unofficial ForgeGradle 2.x Patch

This is an unofficial patch for ForgeGradle 2.1-SNAPSHOT, 2.2-SNAPSHOT and 2.3-SNAPSHOT to accomodate Forge moving their Maven repository from here to there. To see what the hell I actually did, see my GitHub fork.

Since Forge published their own patches, these ones are now obsolete, please use the official patches! Specifically versions 2.3.4, 2.2.2, 2.1.3 and 1.2.2.
